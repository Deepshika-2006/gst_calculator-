# GST Calculator Pro

A modern and responsive GST Calculator built using HTML, CSS, and JavaScript. This application helps users calculate GST accurately using custom tax rates and supports both GST Inclusive and GST Exclusive calculations.

## 📊 How It Works

### GST Exclusive

When the entered amount does not include GST:

Example:

Amount = ₹1000
GST = 18%

GST Amount = ₹180
Total Amount = ₹1180

### GST Inclusive

When the entered amount already includes GST:

Example:

Amount = ₹1180
GST = 18%

GST Amount = ₹180
Original Amount = ₹1000
